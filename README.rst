ORx install scripts
===================

This repository contains a collection of scripts for setting up devices for
receiving and managing Outernet broadcast content.

Err... what's ORx?
==================

ORx is pronouced "Oh-Rex" and stands for Outernet Receiver X. "X" stands for
home-made device.
