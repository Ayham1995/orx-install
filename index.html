Build an Outernet receiver from scratch

The ORx install scripts automate creation of Outernet Receivers based on Raspberry Pi, and using DVB-S tuners.

To get started, follow the instructions for your OS of choice.

Arch Linux ARM Raspbian RaspBMC

This page contains links to resources that are used by the script. These include binaries and other non-source components required for the build that cannot be included in the source repository.

Librarian

Librarian is the archive manager that runs on ORx and provides a web-based user interface for managing the archive. Below are the Python packages for the Librarian. They are platform independent. Librarian requires Python >= 3.3.

Librarian is licensed under GNU GPL v3.
